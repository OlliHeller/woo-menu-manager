## Description (English)

**WooCommerce Menu Manager** allows you to clean up your WordPress admin sidebar by moving up to four WooCommerce top-level menus into the main WooCommerce submenu:

*   Products
*   Payments
*   Analytics
*   Marketing

You can enable or disable each item individually through the settings page. All menu items continue to function exactly as before – nothing is removed, only reorganized for better usability.

### Key Features
*   **Clean Sidebar:** Move clutter into the WooCommerce main tab.
*   **Individual Control:** Choose exactly which items to move.
*   **Lightweight:** No performance overhead or core file modifications.
*   **Multilingual:** Supports Loco Translate, WPML, Polylang, and .po/.mo files.

## Installation

1.  Download the plugin ZIP file.
2.  Upload it via **Plugins → Add New → Upload Plugin**.
3.  Activate the plugin.
4.  Go to **Settings → WC Menu Manager** to configure your sidebar.

## FAQ

**Will this plugin remove any WooCommerce functionality?**  
No. It only reorganizes the menu. All pages continue to work exactly as before.

**Is WooCommerce required?**  
Yes. The plugin only acts if WooCommerce is active.
