Reorganize the WooCommerce admin menu by moving Products, Payments, Analytics, and Marketing into the WooCommerce submenu.

== Description ==

**WooCommerce Menu Manager** helps you clean up your WordPress admin sidebar by moving up to four WooCommerce top-level menus into the main WooCommerce submenu:

* Products
* Payments
* Analytics
* Marketing

You can enable or disable each item individually through the settings page. All menu items continue to function exactly as before — nothing is removed, only reorganized for better usability.

= Key Features =

* **Clean Sidebar:** Move clutter into the WooCommerce main tab.
* **Individual Control:** Choose exactly which items to move.
* **Lightweight:** No performance overhead or core file modifications.
* **Multilingual:** Supports Loco Translate, WPML, Polylang, and .po/.mo files.

== Installation ==

1. Download the plugin ZIP file.
2. Upload it via **Plugins → Add New → Upload Plugin**.
3. Activate the plugin.
4. Go to **Settings → WC Menu Manager** to configure your sidebar.

== Frequently Asked Questions ==

= Will this plugin remove any WooCommerce functionality? =

No. It only reorganizes the menu. All pages continue to work exactly as before.

= Is WooCommerce required? =

Yes. The plugin only acts if WooCommerce is active. If WooCommerce is not installed or deactivated, the plugin will show an admin notice and do nothing.

= Does it support multisite? =

The plugin has not been tested explicitly on multisite setups. Use at your own discretion.
